### 5.5 push 503 vs MCP push timeout

CLAUDE.md §2 提到 git push 被 local proxy 擋（HTTP 503）—— 但本 session 實測 git push 是**可以用的**（成功推 8 次），可能是環境變了。MCP push tools（`mcp__github__push_files` 等）對 >7KB 檔案會 timeout。**先試 git push**，失敗才切 MCP / 手動。

`git push origin --delete <branch>` 還是被擋（403），所以遠端分支刪除目前無法自動，要手動到 GitHub 網頁刪。

### 5.6 沒有測試套件

`package.json` 沒 `npm test`。改 routing 邏輯時用 `/tmp/trace-*.mjs` 寫純 node trace（之前的 PR-5/6 都這樣）；改其他邏輯靠 `npm run build` 通過 + 人類驗證部署網址。
